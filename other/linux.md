# linux 各文件夹说明
>* bin（binary） ：常见linux命令、系统所有用户命令目录文件
>* dev（device） : 设备驱动存储目录文件
>* media: 多媒体及挂载目录
>* proc （process）：进程信息文件
>* sbin（superuser binary） : root用户命令文件
>* var （variable）: 变量文件
>* boot: 引导程序文件
>* etc （etcetera）: 所有程序所需要的配置文件
>* lib （library）: 系统默认库路径文件
>* mnt （mount）: 挂载目录
>* root : root根目录
>* tmp （temporary）: 临时文件目录
>* home: 家目录（用户目录）
>* opt （option）: 可选目录（可以选择安装应用程序的目录）
>* usr （user）: 用户程序目录
